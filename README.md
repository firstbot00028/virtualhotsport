# 🚀 High-Speed Node Proxy Server (Singapore Node)
### Developed by: Engineer Adam 💎

A lightweight, powerful Python-based Proxy Server designed for high-performance data routing. This interface acts as a filtered gateway, capable of handling complex requests with a 150Gbps optimized logic.

---

## 🧠 About the Project
This project is a high-speed proxy utility built using **Flask** and **Requests**. It allows users to route their GET, POST, PUT, and DELETE requests through a secure Singapore-based node simulation. 

Ideal for developers who need a quick proxy setup for API testing or bypassing header restrictions.

## ✨ Features
- **Full Method Support:** Handles GET, POST, PUT, and DELETE.
- **Dynamic Routing:** Supports any HTTPS target URL via path parameters.
- **Header Filtering:** Automatically cleans and forwards essential headers while stripping redundant ones.
- **Streaming Support:** Optimized with `10MB` chunk sizes for fast data transfer.
- **Fail-safe Logic:** Built-in timeout and exception handling to prevent server hangs.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Framework:** Flask
- **Networking:** Requests Library
- **Environment:** Render / Heroku / Local Server
